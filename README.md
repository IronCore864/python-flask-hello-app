# Python Flask Helloworld App

# Dependency

`pip3 install -r requirements.txt`

# Test

'python3 -m unittest discover test'

# Docker

## Hub

https://hub.docker.com/r/ironcore864/flask-hello/

## Build

`make build`

## Run

`make run`

Default port 5000

## Shell

`make shell`

## Stop

`make stop`

## Clean

`make clean`
